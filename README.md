# DNRM Protocol & DREN Infrastructure

Welcome to the core protocol repository for **Do Not Record Me (DNRM)**, a decentralized, edge-computed system for automated visual consent enforcement and threshold-governed biometric recovery.

This repository houses the foundational architecture, the Decentralized Recovery and Escrow Network (DREN) smart contracts, and the official protocol documentation.

## 🛠️ What We Are Building Here (DREN)
We are currently actively developing **Phase 3** of the protocol: The Decentralized Recovery and Escrow Network (DREN). 

Upcoming code commits in this repository will include:
* **Smart Contracts:** Solidity/Rust contracts for the Lawful Recovery Interface.
* **Threshold Cryptography:** Implementations of Shamir's Secret Sharing (SSSS) to shard the Biometric Evidence Envelope (BEE) keys.
* **Node Operator Infrastructure:** Guidelines and node-runner code for independent organizations to participate in DREN consensus.
